# 🎨 IPO Frontend (React)

This is the frontend application for the Bluestock IPO Data Platform, built using React.

It provides a user interface to interact with the backend Django API for managing IPO listings.

---

## 🚀 Features

* 📊 View IPO listings
* ➕ Add new IPO entries
* ✏️ Update IPO details
* ❌ Delete IPO records
* 🔗 Integrated with Django REST API

---

## ⚙️ Tech Stack

* React (Create React App)
* JavaScript
* HTML/CSS
* REST API Integration

---

## 📦 Installation

```bash
npm install
```

---

## ▶️ Run the Application

```bash
npm start
```

👉 App will run at:
http://localhost:3000

---

## 🔗 Backend Connection

Make sure the Django backend server is running:

```bash
http://127.0.0.1:8001
```
